## Features

Every file/COBOL unit test will be open in a seperate tab.
Compile requires GnuCOBOL compiler
Unit tests requires GnuCOBOL compiler and Neopragmas COBOL unit test.

## Requirements

The same COBOL unit test folder as I use.
